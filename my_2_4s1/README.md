# Day1 作业：双四选一数据选择器（Dual 4:1 MUX）

## 作业背景
在完成 Day1 的二选一数据选择器基础练习后，为了进一步巩固：
- 多输入组合逻辑的建模能力
- 选择信号与数据通道之间的对应关系
- 模块结构与代码可读性

我自行设计并实现了本作业：**双四选一数据选择器**。

该作业不依赖现成示例，属于在基础知识之上的主动扩展练习。

---

##  设计实现思路
- 使用 **组合逻辑** 实现，不引入时钟与复位
- 通过 `case(sel)` 描述多路选择关系
- 两路输出逻辑结构保持对称，便于阅读与扩展
- 避免隐式锁存器，确保 `case` 分支完整

---
##  仿真与验证
- 编写 Testbench，对 `sel` 进行 0~3 全遍历
- 同步改变输入数据，验证两路输出是否正确选通
- 重点检查：
  - 是否存在未覆盖的选择状态
  - 仿真波形中是否出现 `X` 或 `Z`
